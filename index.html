UI Framework

Colors

Colors can be defined with CSS variables. They can be reused in the whole UI Framework.

Definition of a CSS variable

In /ui-framework/colors/colors.css:
:host {
  --primary: blue;
}

Usage of a CSS variable

In CSS:
h1 {
  color: var(--primary);
}
